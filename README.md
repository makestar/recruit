## 메이크스타 개발팀 채용 안내 (v2019.7.4)

### 메이크스타는?

세계 '최초' 크라우드펀딩 기반 엔터테인먼트 제작/유통 플랫폼입니다.
2015년 정식 서비스 오픈 이후 누적 프로젝트 180개의 프로젝트를 진행하였으며
92개국 5만여명의 사용자가 펀딩에 참여하였습니다. (2018년 기준)
